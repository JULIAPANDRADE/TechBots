# TechBots
Repositório utilizado para a entrega das atividades realizadas na disciplina "Usina de Projetos Experimentais II" do 2º semestre da Facens, ministrado pela professora Brenda Medina.

Nome do Grupo:
- TechBots

Título:
- prazo.app — Sistema de Controle de Validade e Reaproveitamento de Alimentos (SCVRA)

Objetivo: 
- O SCVRA é uma aplicação digital voltada à gestão doméstica de alimentos, concebida com o propósito de reduzir o desperdício alimentar, otimizar o consumo e apoiar a tomada de decisão do usuário no planejamento alimentar cotidiano, tanto o objetivo geral quanto os específicos. O objetivo geral deve apresentar de forma clara o que o projeto busca alcançar, enquanto os específicos devem detalhar as etapas ou metas menores que ajudam a alcançar o objetivo geral.
